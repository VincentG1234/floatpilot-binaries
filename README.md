# **FloatPilot — Desktop AI Copilot**

FloatPilot is a lightweight floating AI window for macOS and Windows.
Always on top, instant screenshot → preview → send to your OpenAI API key.

This repository hosts the **official binaries only**.
No source code is included.

---

## **Features**

* Always-on-top floating window
* Global shortcut to open instantly
* Screenshot capture with crop & blur
* Bring-your-own OpenAI API key
* No data collection
* Everything runs locally, except API calls
